プライバシーポリシー

AI News Hubの個人情報保護方針

1. はじめに

AI News Hub（以下「当サイト」）は、学習目的で作成されたAI関連ニュース集約サイトです。当サイトでは、ユーザーのプライバシーを尊重し、適切な個人情報保護に努めております。

重要：当サイトは教育・学習目的で作成されており、商用利用は行っておりません。

2. 収集する情報

2.1 自動的に収集される情報

  • IPアドレス
  • ブラウザの種類とバージョン
  • オペレーティングシステム
  • アクセス日時
  • 閲覧ページ情報
  • リファラー情報

2.2 ユーザーが提供する情報

当サイトでは、現在のところユーザー登録機能やコメント機能は実装しておりません。そのため、ユーザーが意図的に提供する個人情報の収集は行っておりません。

2.3 Cookie（クッキー）

当サイトでは、ユーザー体験の向上のため、以下の目的でCookieを使用する場合があります：

  • サイトの設定情報の保存（言語設定、テーマ設定など）
  • アクセス解析のための匿名データ収集

3. 情報の利用目的

収集した情報は以下の目的で利用されます：

  • サイトの運営・管理
  • サービス品質の向上
  • 技術的問題の解決
  • アクセス統計の作成（匿名化）
  • セキュリティの確保

4. 第三者サービスの利用

4.1 外部API

当サイトでは、ニュース情報の取得のため以下の外部APIサービスを利用しています：

  • News API - ニュース記事の取得
  • YouTube Data API - 動画情報の取得
  • Twitter API - ソーシャルメディア情報の取得
  • Google Trends API - 検索トレンド情報の取得

これらのサービスの利用に際して、各サービスのプライバシーポリシーが適用される場合があります。

4.2 外部リンク

当サイトには外部サイトへのリンクが含まれています。これらの外部サイトのプライバシーポリシーについては、当サイトは責任を負いません。

5. 情報の共有・開示

当サイトでは、以下の場合を除き、収集した情報を第三者に提供することはありません：

  • 法令に基づく場合
  • ユーザーの生命、身体または財産の保護のために必要がある場合
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合
  • 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合

6. 情報の保護

当サイトでは、収集した情報の漏洩、滅失または毀損の防止その他の安全管理のため、必要かつ適切な措置を講じるよう努めています。

セキュリティ対策：HTTPS通信の使用、適切なアクセス制御、定期的なセキュリティ更新を実施しています。

7. 学習目的サイトに関する免責事項

当サイトは学習・教育目的で作成されており、以下の点にご注意ください：

  • 提供される情報の正確性や完全性を保証するものではありません
  • 商用利用や営利目的での使用は想定されていません
  • サイトの継続的な運営を保証するものではありません
  • 技術的な実験や学習のため、予告なく機能変更が行われる場合があります

8. プライバシーポリシーの変更

当サイトでは、法令の変更や当サイトの機能向上に伴い、本プライバシーポリシーを変更する場合があります。重要な変更については、当サイト上での告知を行います。
